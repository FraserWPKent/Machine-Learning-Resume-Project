            Matplotlib for image display
            pytorch for machine learning framework
            Image Data set from either kraggle or a personally curated one: www.kaggle.com/datasets/kaustubhdhote/human-faces-dataset/data

            The model should be able to take in a image and reliably > 90-95% tell if the image is of a real human or a ai generated faximily of

            Perhaps build in a process than automatically requests images from a modern model like open ai, or gemini to get more modern data. (Least important)

        Learning Goals:
            Understand machine learning
            Understand the process of image processing
            Understand the process of dataset building, cleaning, and organizing.
            Get a more indept undestanding of python
